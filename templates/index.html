📖 API Reference

Important: Please replace SERVICE_URL in the examples below with the official URL provided by your deployment (e.g., Cloud Run or Localhost).
GET /health
Diagnostic check for server and model status.
Example:
curl -X GET SERVICE_URL/health
POST /predict
Submits 20 features to the ML pipeline. Expects a JSON object nested in a "data" key.
LINUX / APPLE / POWERSHELL
curl -X POST "SERVICE_URL/predict" -H "Content-Type: application/json" -d '{"data": {"id": "7229300521", "date": "20141013T000000", "bedrooms": 2, "bathrooms": 1, "sqft_living": 1180, "sqft_lot": 5650, "floors": "1", "waterfront": 0, "view": 0, "condition": 3, "grade": 7, "sqft_above": 1180, "sqft_basement": 0, "yr_built": 1955, "yr_renovated": 0, "zipcode": "98178", "lat": 47.5112, "long": -122.257, "sqft_living15": 1340, "sqft_lot15": 5650}}'
WINDOWS (CMD)
curl -X POST "SERVICE_URL/predict" -H "Content-Type: application/json" -d "{\"data\": {\"id\": \"7229300521\", \"date\": \"20141013T000000\", \"bedrooms\": 2, \"bathrooms\": 1, \"sqft_living\": 1180, \"sqft_lot\": 5650, \"floors\": \"1\", \"waterfront\": 0, \"view\": 0, \"condition\": 3, \"grade\": 7, \"sqft_above\": 1180, \"sqft_basement\": 0, \"yr_built\": 1955, \"yr_renovated\": 0, \"zipcode\": \"98178\", \"lat\": 47.5112, \"long\": -122.257, \"sqft_living15\": 1340, \"sqft_lot15\": 5650}}"
GET /past_predictions
Retrieves the history of all predictions stored in memory.
Example:
curl -X GET SERVICE_URL/past_predictions
PUT /past_predictions/{id}
Updates an existing record's price prediction by its ID.
LINUX / APPLE / POWERSHELL
curl -X PUT "SERVICE_URL/past_predictions/7229300521" -H "Content-Type: application/json" -d '{"prediction": 250000.0}'
WINDOWS (CMD)
curl -X PUT "SERVICE_URL/past_predictions/7229300521" -H "Content-Type: application/json" -d "{\"prediction\": 250000.0}"
