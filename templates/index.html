🚀 Execution Guide

Choose the command based on your operating system:
Local Development (uv)
uv run uvicorn src.api.app:app --host 0.0.0.0 --port LOCAL_PORT --reload
URL: http://localhost:LOCAL_PORT
Docker Container
docker run -p LOCAL_PORT:8080 valore-app
URL: http://localhost:LOCAL_PORT
Port Configuration: Replace LOCAL_PORT with a valid port number (e.g., 8000). Ensure the port is open on your firewall and not in use.
Google Cloud Run (Deploy) WINDOWS (CMD)
gcloud run deploy valore --project=directed-work-lab2 --region=europe-west1 --source="%cd%" --allow-unauthenticated
LINUX / APPLE / POWERSHELL
gcloud run deploy valore --project=directed-work-lab2 --region=europe-west1 --source=$(pwd) --allow-unauthenticated
URL: SERVICE_URL

📖 API Reference (Replace URL with your target)

GET /health
Diagnostic check for server and model status.
Example:
curl -X GET URL/health
POST /predict
Submits 20 features to the ML pipeline. Expects a JSON object nested in a "data" key.
LINUX / APPLE / POWERSHELL
curl -X POST "URL/predict" -H "Content-Type: application/json" -d '{"data": {"id": "7229300521", "date": "20141013T000000", "bedrooms": 2, "bathrooms": 1, "sqft_living": 1180, "sqft_lot": 5650, "floors": "1", "waterfront": 0, "view": 0, "condition": 3, "grade": 7, "sqft_above": 1180, "sqft_basement": 0, "yr_built": 1955, "yr_renovated": 0, "zipcode": "98178", "lat": 47.5112, "long": -122.257, "sqft_living15": 1340, "sqft_lot15": 5650}}'
WINDOWS (CMD)
curl -X POST "URL/predict" -H "Content-Type: application/json" -d "{\"data\": {\"id\": \"7229300521\", \"date\": \"20141013T000000\", \"bedrooms\": 2, \"bathrooms\": 1, \"sqft_living\": 1180, \"sqft_lot\": 5650, \"floors\": \"1\", \"waterfront\": 0, \"view\": 0, \"condition\": 3, \"grade\": 7, \"sqft_above\": 1180, \"sqft_basement\": 0, \"yr_built\": 1955, \"yr_renovated\": 0, \"zipcode\": \"98178\", \"lat\": 47.5112, \"long\": -122.257, \"sqft_living15\": 1340, \"sqft_lot15\": 5650}}"
GET /past_predictions
Retrieves the history of all predictions stored in memory.
Example:
curl -X GET URL/past_predictions
PUT /past_predictions/{id}
Updates an existing record's price prediction by its ID.
LINUX / APPLE / POWERSHELL
curl -X PUT "URL/past_predictions/7229300521" -H "Content-Type: application/json" -d '{"prediction": 250000.0}'
WINDOWS (CMD)
curl -X PUT "URL/past_predictions/7229300521" -H "Content-Type: application/json" -d "{\"prediction\": 250000.0}"
